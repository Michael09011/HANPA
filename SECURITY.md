# 보안 보고

HANPA 프로젝트의 보안 문제를 발견해 주셔서 감사합니다. 보안 취약점은 신속하게 확인하고 해결하려고 합니다.

## 보고 방법

1. GitHub 이슈로 취약점을 보고하세요. 보고 시 재현 단계, 영향 범위, 취약점 세부 정보를 포함해 주세요.
2. 공개 보고가 어려운 경우, 저장소 소유자에게 GitHub DM이나 이메일로 연락해 주세요.

## 응답 원칙

- 보고된 문제를 확인한 뒤 가능한 한 빠르게 답변드립니다.
- 보안 이슈는 필요 시 먼저 수정하고 공개적으로 알릴 수 있습니다.
- 취약점 공개는 프로젝트 안정성과 안전을 우선으로 합니다.

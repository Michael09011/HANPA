# TECH

이 폴더는 HANPA 프로젝트 자체의 기술 문서를 정리하는 공간입니다.

## 용도

- HANPA의 아키텍처, Electron 통합, UI/렌더러 설계
- `@rhwp/core` 엔진과 HANPA 간 연동 방식
- HANPA 고유 기능 개발과 기술 결정 배경

## 참고

- 엔진 연구 및 `rhwp` 관련 기술 문서는 `third_party/rhwp/mydocs/tech/`에 보관됩니다.
- 이 폴더는 HANPA 자체 기술 문서를 별도로 정리할 때 사용합니다.

## HANPA TECH 문서

- `tech/electron.md` — Electron 기반 데스크톱 앱 아키텍처
- `tech/byte-handling.md` — HWP 파일 및 이진 데이터 처리
- `tech/typescript.md` — TypeScript 사용 이유 및 구성
